# Thomas Newman CV AI

## Pre-requisites

This project uses the following technologies:

- [pnpm](https://pnpm.io/)
- [Turborepo](https://turborepo.com/)
- [Node.js](https://nodejs.org/)

Install Turborepo globally:

```bash
pnpm add turbo --global
```

## Setup

Install dependencies:

```bash
pnpm install
```
