**October 2023 - November 2024**

Frontend Lead for the Commercial Banking Online website.

## Tech Stack

- React
- TypeScript
- Module Federation 2.0
- Webpack
- ESLint
- Stylelint
- Testing Library
- Mock Service Worker
- Zustand
- Jotai
- Legend State
- Redux Toolkit
- Figma

## Responsibilities

### Architectural Leadership

Leading the micro front ends approach for migration of the site to module federation

### Component Library

Leading the implementation of a component library

### Design System Implementation

Working closely with designers to ensure consistency with a new design system and theming in Figma

### Mentoring

Mentoring junior engineers and demonstrating best practices

## Key Achievements

### Codebase Optimization

Refactored codebase to speed up the delivery of features and improved webpack configurations

### Micro Frontend POC

Created a POC for micro front ends, demonstrating certain approaches for managing shared dependencies, error boundaries and routing

### Risk Mitigation

Highlighted areas requiring risk mitigation in order to safely support module federation development, testing and deployment strategies

### Legacy Migration

Decoupled and migrated old React SPAs to Module Federation 2.0 with updated linting and fixed all TypeScript issues

### Testing Infrastructure

Created a new Playwright acceptance test setup to mitigate runtime risks and complexity so issues are caught earlier in a pipeline

### Testing Best Practices

Demonstrated best practices for unit using mock service worker and testing library

### Knowledge Sharing

Created a Frontend Guild for knowledge sharing and Brown Bag sessions
