# Senior Frontend Engineer at Dayrize

**November 2024 - June 2025** _(Redundancy due to insolvency)_

## Company Context

Sole Frontend ownership of the Dayrize application in a startup environment. The web application is a data analytics platform which includes data graphs, tables with filtering, data ingestion, data edit and file upload.

## Tech Stack

- React
- TypeScript
- ESLint
- Testing Library
- Mock Service Worker
- Vite
- React Router
- Tanstack Query/Form/Table
- Mantine
- CSS Modules
- BitBucket
- Google Cloud
- Cursor AI

## Responsibilities

### Technical Leadership

Fully responsible for the Frontend technical direction and testing strategy

### Feature Delivery

Delivering features and bug fixes at high quality and speed

### Design

UX and UI design and implementation

### Maintenance & Upgrades

Updating all Node packages and migration approaches

### Stakeholder Communication

Company presentations/demos of new and upcoming features

## Key Achievements

### UX Enhancement

Introduced optimistic UI to the platform to reduce UI blocking functionality, improving the user experience

### Performance Fixes

Fixed React component composition and state management that was causing stuttering and slowdown to the UX

### Testing Optimization

Refined the unit tests by following best practices and cut the pipeline time down by a third

### Code Consistency

Centralized the styling away from various inline styling approaches to CSS modules

### Performance Improvements

Improved application performance by introducing code splitting and moving API calls up the HTTP waterfall using loaders

### Process Improvements

Moved work process towards user-focused story writing and introduced refinement sessions

### Asset Optimization

Added compression to bundled assets on Google Bucket

### Code Standards

Introduced linting practices ensuring standards are followed for various libraries such as Tanstack and React

### TypeScript Excellence

Updated the TypeScript setup for best practices and fixed all the errors

### Data Ingestion

Integrated data ingestion feature with Flatfile

### Advanced Tables

Introduced higher performing tables with filtering, sorting and search with abort controllers

### Feature Design & Implementation

Designed and created the UX/UI for:

- Data share platform for sending, receiving, reviewing data sharing requests
- Login journey for 2 factor authentication with QR code
- Data ingestion journeys with integration with Flatfile
