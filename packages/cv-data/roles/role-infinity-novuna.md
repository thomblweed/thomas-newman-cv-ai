# Full Stack Engineer Consultant at Infinity Works - Novuna

**June 2021 - March 2022**

## Project Context

Working in one of 3 agile feature teams and being part of delivering the new live version of a digital application, Mercury

- <https://www.novuna.co.uk/business-finance/mercury-hub>
- <https://www.mercuryhg.co.uk>

## Tech Stack

- React
- TypeScript
- Node (AWS Lambda/Middy)
- Serverless architecture
- Testing Library
- Testcafe with Gherkin
- Material UI

## Responsibilities

### Serverless Development

Writing clean code in TypeScript, React, Node (AWS Lambda/Middy) for a serverless application that communicates with an in-house API Gateway

### Mentoring

Supporting the company junior engineers and consulting the client engineers on best practices

### Feature Delivery

Delivering MVP features ready for the application go live using vertically sliced user stories

### Refinement

Consistent user stories refinement with the business PO and technical refinement with the teams' engineers

### Testing

Writing automated feature/acceptance tests using Gherkin wrapped testcafe

### Rebrand Delivery

Delivering an application styling rebrand within a 4-month deadline that required a single switch on mechanism

## Key Achievements

### Technical Leadership

Demonstrated to team members on benefits of good state management, effective unit testing with Testing Library, refining of coupled code that is not so easily extendible and the long-term effects

### Rebrand Engineering

Partially lead the engineering approach for the rebrand of the site styling, which needed to be done in parallel with the current brand until the go live on the brand switch over deadline

### Performance Optimization

Performed an application workflow update to improve the user journey performance and decoupled a lot of state management

### Library Upgrades

Upgraded the Material UI library from v4 to v5 and demonstrated the benefits of the upgrade to the teams

### Quick Adaptation

Picked up TypeScript quickly after a 3-year commercial hiatus and React after 1 year hiatus
