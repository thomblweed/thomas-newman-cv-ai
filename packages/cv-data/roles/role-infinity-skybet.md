# Full Stack Engineer Consultant at Infinity Works - Sky Bet

**April 2022 - June 2022**

## Project Context

Working in an agile team for the Sporting Life website - <https://www.sportinglife.com/>

## Tech Stack

- JavaScript, React, Next.js, Node
- Java (Dropwizard)
- PostgreSQL on AWS RDS
- Kubernetes (AWS EKS)
- AWS Kinesis, SQS, S3, Cognito
- Testcafe, Enzyme

## Responsibilities

### Full Stack Development

Writing clean code in JavaScript, React, Next.js, Node, Java (Dropwizard) for a SSR website driven by data feeds (AWS Kinesis, SQS) and an in-house CMS which is updated by content editors

### Database Work

Working with databases in Postgres running on AWS RDS and Kubernetes (AWS EKS) that supports the Continuous Delivery of the site

### CMS Development

Developing features and fixes to the in-house CMS (React, Java) which is coupled with AWS S3 and Cognito

### Testing

Writing automated acceptance tests using testcafe and unit tests using Enzyme

## Key Achievements

### Legacy System Revival

Enabled and documented a development environment for 2 legacy Java APIs to unlock the ability to support them (previously not touched in 4 years) and provided some overdue bug fixes for the client

### Testing Initiative

Started promoting the importance of testing to the team and driving the initiative to elevate a large gap in the automated testing area

### Testing Library Migration

Currently promoting to switch over to Testing Library for testing going forwards

### Process Improvement

Promoting the idea of vertical sliced stories instead of horizontal technical tasks and the value this can provide to the client and our ways of working and collaboration
